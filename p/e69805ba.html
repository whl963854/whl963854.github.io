    4.4 жӢүеҸ–д»Јз Ғ
    $ git pull вҖ“rebaseжӢүеҸ–иҝңзЁӢд»“еә“зҡ„д»Јз ҒжҺЁиҚҗдҪҝз”ЁдёҠйқўзҡ„е‘Ҫд»ӨпјҢжҺҘдёӢжқҘдјҡиҜҙжҳҺдёәд»Җд№ҲгҖӮ
    4.4.1 git fetch
    git fetchе‘Ҫд»Өдјҡд»ҺиҝңзЁӢд»“еә“еҸ–еӣһжүҖжңүеҲҶж”Ҝзҡ„жӣҙж–°пјҢдҪҶдёҚдјҡдҝ®ж”№жң¬ең°е·ҘдҪңзӣ®еҪ•зҡ„еҶ…е®№пјҢжӢүеҸ–еҗҺзҡ„жӣҙж–°йңҖиҰҒжң¬ең°еҶҚеҗҲе№¶гҖӮеҰӮдёӢеӣҫжүҖзӨәпјҢиҝҗиЎҢgit fetchеҗҺ HEAD жҢҮй’Ҳе№¶жІЎжңүеҸҳеҢ–гҖӮ
    иҝҷж—¶еҖҷиҝҗиЎҢgit statusдјҡжҸҗзӨәеҗҲе№¶пјҢдҪҝз”Ёgit mergeжҲ–иҖ…git rebaseеҸҜд»ҘеҗҲе№¶д»Јз ҒгҖӮ
    4.4.2 git pull
    дҪҝз”Ё fetch жӢүеҸ–д»Јз ҒеҰӮжӯӨз№ҒзҗҗпјҢиҮӘз„¶е°ұдјҡжңүз®ҖеҢ–зҡ„е‘Ҫд»ӨгҖӮgit pullеҸҜд»ҘеҸ–еӣһжҹҗдёӘеҲҶж”Ҝзҡ„жӣҙж–°е№¶дёҺжң¬ең°зҡ„еҲҶж”ҜеҗҲе№¶пјҢеңЁеӨ§еӨҡж•°жғ…еҶөдёӢе®ғзӯүдәҺgit fetch + git mergeгҖӮ
    git pull дҪҝз”Ё merge иҝӣиЎҢиҮӘеҠЁеҗҲе№¶пјҢеҰӮжһңжң¬ең°жңүжңӘжҺЁйҖҒзҡ„жҸҗдәӨдё”дә‘з«Ҝжңүжӣҙж–°пјҢgit pull дјҡиҮӘеҠЁеҗҲе№¶д»Јз ҒпјҢиҝҷдјҡеңЁжҸҗдәӨеҺҶеҸІдёҠз”ҹжҲҗдёҖдёӘеҗҲе№¶иҠӮзӮ№гҖӮжҜ«ж— з–‘й—®пјҢиҝҷжҳҺжҳҫжұЎжҹ“дәҶеҺҶеҸІи®°еҪ•зҡ„ж•ҙжҙҒгҖӮдёӢеӣҫеҫҲжҳҺжҳҫең°иҜҙжҳҺдәҶиҝҷдёҖзӮ№пјҢиҝҷж ·зҡ„еҗҲе№¶иҠӮзӮ№еҜ№еҺҶеҸІи®°еҪ•жІЎжңүд»»дҪ•дҪңз”ЁпјҢеӣ дёәжҲ‘д»¬жң¬иә«е№¶жІЎжңүеҜ№еҲҶж”ҜиҝӣиЎҢдәҶеҗҲе№¶ж“ҚдҪңгҖӮ
    е®һйҷ…ејҖеҸ‘иҝҮзЁӢдёӯпјҢдёҚжҺЁиҚҗдҪҝз”Ё git pull жӣҙж–°д»Јз ҒпјҢдҪҶд»Ӣз»ҚдёҖз§ҚеҸҜд»ҘдҪҝз”Ё git pull зҡ„жғ…еҶөпјҡ
    п»ҝеҰӮжһңдҪҝз”Ё git merge еҗҲе№¶дәҶдёҖдёӘеҲҶж”ҜпјҢжҺЁйҖҒд»Јз Ғзҡ„ж—¶еҖҷеҸ‘зҺ°иҝңзЁӢд»“еә“жңүжӣҙж–°еҜјиҮҙйңҖиҰҒе…Ҳжӣҙж–°жң¬ең°д»Јз ҒпјҢиҝҷж—¶еҰӮжһңдҪҝз”Ё rebase жӣҙж–°д»Јз ҒпјҢдјҡеҜјиҮҙ git merge иҮӘеҠЁз”ҹжҲҗзҡ„ merge иҠӮзӮ№дёўеӨұгҖӮз”ұдәҺ WGS жҳҜж №жҚ®жңҖеҗҺдёҖдёӘжҸҗдәӨзҡ„ commit id жқҘеҲӨж–ӯжҳҜеҗҰиғҪе…Ҙеә“зҡ„пјҢеҰӮжһң merge иҠӮзӮ№дёўеӨұпјҢдјҡеҜјиҮҙж— жі•е…Ҙеә“йңҖиҰҒйҮҚж–°е®Ўжү№гҖӮеҜ№дәҺиҝҷз§Қжғ…еҶөпјҢеҸҜд»ҘдҪҝз”Ё git pull жӣҙж–°д»Јз ҒпјҲеҜ№дәҺйҮ‘еұұзҡ„д»Јз Ғе№іеҸ°жқҘиҜҙпјҢgit pull иҮӘеҠЁеҗҲе№¶з”ҹжҲҗзҡ„иҠӮзӮ№жҳҜеҸҜд»Ҙе…Ҙеә“зҡ„пјҢдёҚйңҖиҰҒйҮҚж–°жҸҗдәӨд»Јз Ғз”іиҜ·пјүгҖӮ

    Git зҡ„е·ҘдҪңжөҒе’Ңж–Үд»¶зҠ¶жҖҒдёҖиҲ¬еҰӮдёӢеӣҫжүҖзӨәпјҡ
    5.1 жҹҘзңӢд»Јз Ғж”№еҠЁ

    8. еҺҶеҸІи®°еҪ•

    вҖў дёҖдёӘжҸҗдәӨзҡ„ж”№еҠЁдёҚеә”иҜҘиҝҮеӨҡгҖӮиҝҮеӨҡеҸҜиғҪж„Ҹе‘ізқҖиҝҷдёӘжҸҗдәӨеҒҡдәҶеӨӘеӨҡзҡ„дәӢжғ…пјҢд№ҹи®ёеә”иҜҘиҖғиҷ‘жӢҶеҲҶжҸҗдәӨгҖӮ

    еңЁеҹәзЎҖе‘Ҫд»ӨйҮҢйқўзЁҚеҫ®д»Ӣз»ҚдәҶдёҖдёӢ log е‘Ҫд»ӨпјҢдёҖиҲ¬жҲ‘д»¬дҪҝз”Ёgit logжҹҘзңӢеҺҶеҸІи®°еҪ•пјҢlog е‘Ҫд»ӨжңүеҫҲеӨҡеҸӮж•°еҸҜд»Ҙе®ҡеҲ¶еҢ–иҫ“еҮәж јејҸгҖӮејҖеҸ‘иҝҮзЁӢдёӯе»әи®®дҪҝз”ЁеҸҜи§ҶеҢ–е·Ҙе…·жҹҘзңӢж—Ҙеҝ—пјҢеӣ дёәиҝҷиғҪеҗҢж—¶зңӢеҲ°д»Јз Ғж”№еҠЁиҜҰжғ…пјҢжүҖд»ҘиҝҷйҮҢдёҚдјҡиҝҮеӨҡд»Ӣз»Қ log зҡ„еҸӮж•°пјҢиҖҢжҳҜдјҡд»Ӣз»ҚеҰӮдҪ•жӣҙеҘҪең°еҖҹеҠ© log её®еҠ©ејҖеҸ‘е’ҢеҰӮдҪ•и®© log дҝқжҢҒж•ҙжҙҒгҖӮ
    8.1 жҹҘзңӢеӯ—з¬ҰдёІж”№еҠЁ

    8.2 жҹҘзңӢдёҖдёӘж–Үд»¶жҳҜдҪ•ж—¶иў«еҲ йҷӨзҡ„

    жҹҘзңӢдёҖдёӘж–Үд»¶жҲ–зӣ®еҪ•зҡ„еҺҶеҸІи®°еҪ•

    8.3 Git еј•з”Ёж—Ҙеҝ—
    $ git reflogеҪ“дҪ еңЁе·ҘдҪңж—¶пјҢGit дјҡеңЁеҗҺеҸ°дҝқеӯҳдёҖдёӘеј•з”Ёж—Ҙеҝ—пјҲreflogпјүпјҢеј•з”Ёж—Ҙеҝ—и®°еҪ•дәҶ HEAD е’ҢеҲҶж”Ҝеј•з”ЁжүҖжҢҮеҗ‘зҡ„еҺҶеҸІгҖӮGit дјҡй»ҳй»ҳең°и®°еҪ•жҜҸдёҖж¬Ўж”№еҸҳ HEAD ж—¶е®ғзҡ„еҖјгҖӮжҜҸдёҖж¬ЎжҸҗдәӨжҲ–ж”№еҸҳеҲҶж”ҜпјҢеј•з”Ёж—Ҙеҝ—йғҪдјҡиў«жӣҙж–°гҖӮйҖҡиҝҮдҪҝз”Ёеј•з”Ёж—Ҙеҝ—пјҢеҸҜд»ҘжҒўеӨҚдёўеӨұзҡ„жҸҗдәӨгҖӮ
    еј•з”Ёж—Ҙеҝ—еҸӘеӯҳеңЁдәҺжң¬ең°д»“еә“пјҢе®ғеҸӘжҳҜдёҖдёӘи®°еҪ•дҪ еңЁиҮӘе·ұзҡ„д»“еә“йҮҢеҒҡиҝҮд»Җд№Ҳзҡ„ж—Ҙеҝ—гҖӮеј•з”Ёж—Ҙеҝ—зҡ„ж•°жҚ®еӯҳж”ҫеңЁ.git/logs/зӣ®еҪ•дёӯгҖӮ

    8.5 и°ғж•ҙеҺҶеҸІи®°еҪ•
    еңЁејҖеҸ‘иҝҮзЁӢдёӯпјҢеҫҲеӨҡж—¶еҖҷжҲ‘д»¬йғҪеҸҜиғҪйңҖиҰҒдҝ®и®ўеҺҶеҸІи®°еҪ•пјҢд№ҹи®ёеҸӘжҳҜжҸҗдәӨдҝЎжҒҜеҶҷй”ҷдәҶпјҢжҲ–жҳҜеӨҡжқЎи®°еҪ•йҮҢйқўзҡ„жҹҗдёҖжқЎи®°еҪ•зҡ„д»Јз Ғжңүй—®йўҳжғіиҰҒйҮҚж–°дҝ®ж”№пјҢеҸҲжҲ–жҳҜдёҖжқЎжҸҗдәӨйҮҢйқўд»Јз ҒеӨӘеӨҡдәҶжғіиҰҒжӢҶеҲҶгҖӮеңЁд»Јз ҒжҸҗдәӨеҲ°иҝңзЁӢд»“еә“д№ӢеүҚпјҢжҲ‘д»¬йғҪеҸҜд»ҘеңЁжң¬ең°йҡҸдҫҝдҝ®ж”№еҺҶеҸІи®°еҪ•гҖӮз„¶иҖҢдёҖж—Ұе°Ҷд»Јз ҒжҺЁйҖҒеҲ°дәҶиҝңзЁӢд»“еә“пјҢйӮЈд№ҲйҷӨйқһзү№ж®Ҡжғ…еҶөпјҢйғҪдёҚе»әи®®зӣҙжҺҘдҝ®ж”№иҝңзЁӢд»“еә“зҡ„еҺҶеҸІи®°еҪ•гҖӮз®ҖиҖҢиЁҖд№ӢпјҢжҲ‘д»¬еә”иҜҘйҮҚи§ҶеҺҶеҸІи®°еҪ•пјҢеҰӮжһңеҺҶеҸІи®°еҪ•жңүй—®йўҳпјҢе°ұдёҚеә”иҜҘе°Ҷд»Јз ҒжҺЁйҖҒеҲ°иҝңзЁӢд»“еә“гҖӮ

    8.5.2 дҝ®ж”№еӨҡдёӘжҸҗдәӨ
